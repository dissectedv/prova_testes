# Projeto de Testes Automatizados com Cypress - Sauce Demo

## 📌 Sobre
Este projeto contém testes automatizados E2E da aplicação [Sauce Demo](https://www.saucedemo.com/) utilizando o Cypress.

## 🛠 Pré-requisitos
- Node.js instalado
- npm ou yarn

## ⚙️ Instalação

```bash
npm install
```

## ▶️ Rodando os testes

```bash
npx cypress open
```

Ou em modo headless:

```bash
npx cypress run
```
